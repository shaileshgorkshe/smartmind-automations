# SmartMind Automations — Next.js Netlify Starter


## Quick start
1. Clone this repo.
2. `npm install`
3. `npm run build` (produces `out/` folder)
4. Push the repository to GitHub.
5. Connect the repo to Netlify via **New site → Import from Git**.
- Build command: `npm run build`
- Publish directory: `out`
6. After deploy, enable **Identity** and **Git Gateway** in Netlify settings.
7. In the Netlify site dashboard, go to **Site settings → Identity** → Enable Identity. Then under **Identity → Services** enable Git Gateway.
8. Visit `https://<yoursite>.netlify.app/admin` to log in and manage content.
